Spark Mailing Lists

Get help using Spark or contribute to the project on our mailing lists:

  • user@spark.incubator.apache.org is for usage questions, help, and announcements. (subscribe) (archives)
  • dev@spark.incubator.apache.org is for people who want to contribute code to Spark. (subscribe) (archives)

User Meetups

We host a regular Bay Area Spark User Meetup where you can meet the community in person. The meetup.com page also contains an archive of past meetups, and slides from them.

Issue Tracker

The project tracks bugs and new features on JIRA. If you'd like, you can also subscribe to the spark-issues group to receive emails about new or updated issues.

Old Mailing Lists

The following Google Groups contain Spark usage and developer discussions from before the transition to Apache. They will become read-only in September 2013:

  • spark-users -- usage questions, help, and announcements.
  • spark-developers -- code contributor discussion.

Spark is an open source project developed at the UC Berkeley AMPLab.
