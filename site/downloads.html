Download Spark

The latest release is Spark 0.9.0, released February 2, 2014 (release notes) (git tag)

  • Download sources: find an Apache mirror or direct file download
  • Download binaries for Hadoop 1 (HDP1, CDH3): find an Apache mirror or direct file download
  • Download binaries for CDH4: find an Apache mirror or direct file download
  • Download binaries for Hadoop 2 (HDP2, CDH5): find an Apache mirror or direct file download
  • Verify your download: signatures and checksums

The Spark download includes Spark Streaming, MLlib and GraphX (alpha).

Link with Spark

Spark artifacts are hosted in Maven Central. You can add a Maven dependency with the following coordinates:

groupId: org.apache.spark
artifactId: spark-core_2.10
version: 0.9.0-incubating

Development Version

If you are interested in working with the newest under-development code or contributing to Spark development, you can also check out the master branch from Git: git clone git://github.com/apache/incubator-spark.git.
