--- layout: post title: Spark 0.7.0 released categories: - News tags: [] status: publish type: post published: true meta: _edit_last: '4' _wpas_done_all: '1' --- We're proud to announce the release of Spark 0.7.0, a new major version of Spark that adds several key features, including a Python API for Spark and an alpha of Spark Streaming. This release is the result of the largest group of contributors yet behind a Spark release -- 31 contributors from inside and outside Berkeley. Head over to the release notes to read more about the new features, or download the release today.
